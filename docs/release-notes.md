# リリースノート

## バージョン 1.0.0

はじめてのリリースです。出会った人のことを、もっと覚えていられるアプリ「Recall」をお届けします。

### 主な機能

- **人物管理** — 名前・所属・関係性・趣味・出身地・写真をまとめて記録できます
- **会話ログ** — 会った日時・場所・話した内容を日記のように残せます
- **タグで整理** — 「同期」「取引先」など自由なタグで人物をグループ管理できます
- **リマインダー** — 「次に会う日」を登録すると、前日夜に通知でお知らせします
- **予定登録** — 会う予定を登録して、前日に通知を受け取れます
- **バックアップ** — データをJSON・CSVで書き出して安全に保管できます
- **完全オフライン** — すべてのデータはデバイス内に保存。ログイン不要で使えます
