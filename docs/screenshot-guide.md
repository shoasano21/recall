# スクリーンショット撮影ガイド

## 必要なサイズ

| デバイス | 解像度 | 備考 |
|---------|--------|------|
| 6.9インチ（iPhone 16 Pro Max）| 1320 × 2868 px | **必須** |
| 6.5インチ（iPhone 14 Plus など）| 1284 × 2778 px | 推奨 |

> App Store Connect では最大10枚まで登録できます。最低1枚（6.9インチ）が必須です。

---

## 撮影手順（Windows + 実機 Expo Go）

### 事前準備

1. iPhone に **Expo Go** をインストール（App Store で無料）
2. PC と iPhone を **同じ Wi-Fi** に接続
3. ターミナルで起動：
   ```bash
   npx expo start --clear
   ```
4. Expo Go でQRコードをスキャン → アプリが起動

### スクリーンショットの撮り方

| iPhone モデル | 操作 |
|-------------|------|
| Face ID 搭載（iPhone X以降）| サイドボタン + 音量上ボタン を同時押し |
| Touch ID 搭載（iPhone SE など）| ホームボタン + サイドボタン を同時押し |

撮影した画像は **写真アプリ** に保存されます。
PCへの転送：ケーブル接続 → Windowsエクスプローラーから `DCIM` フォルダを開いてコピー。

### 撮影後の保存先

`docs/screenshots/` フォルダに以下のファイル名で保存してください：

```
docs/screenshots/
  01_splash.png
  02_home.png
  03_detail.png
  04_add_person.png
  05_conversation_log.png
```

## 推奨シーン（5枚）

### Scene 1 — スプラッシュ画面 → `01_splash.png`

**撮影方法**
- Expo Go でアプリを完全終了（ホーム画面でアプリを上にスワイプして終了）
- 再起動して、スプラッシュ（青グラデーション＋「Recall」）が表示された瞬間に撮影

**表示内容**
- 青グラデーション背景
- 「Recall」タイトル
- キャッチコピー「大切な人のことを、もっと覚えていよう」

---

### Scene 2 — ホーム画面 → `02_home.png`

**表示内容**
- 4人の人物カード（田中 誠・佐藤 美咲・鈴木 健太・山本 花）
- タグフィルター（ビジネス・取引先・同期・大学・上司・友人・プライベートが並ぶ）
- 検索バー・右下FABボタン

**撮影のコツ**
- タグフィルターを「すべて」のままにして全員表示
- またはタグ「ビジネス」で絞り込んだ状態も映える

---

### Scene 3 — 人物詳細画面 → `03_detail.png`

**撮影方法**
- ホームから「田中 誠」をタップ

**表示内容**
- 名前・所属・関係性・タグ
- 趣味・出身地
- 会話ログ2件（3日前・30日前）
- 右下の＋FABボタン

---

### Scene 4 — 人物追加画面 → `04_add_person.png`

**撮影方法**
- 右下FABボタンをタップ → 人物追加モーダルを開く
- 以下のデータを入力した状態で撮影：

| フィールド | 入力値 |
|----------|--------|
| 名前 | 伊藤 大輝 |
| 所属 | ○○株式会社 |
| 関係性 | 同僚 |
| タグ | 同僚 |

**撮影のコツ**
- フォームの上半分（名前・所属・関係性・タグ）が見える位置で撮る

---

### Scene 5 — 会話ログ追加画面 → `05_conversation_log.png`

**撮影方法**
- 人物詳細画面の右下＋ボタン → 「会話を追加」をタップ
- 以下の内容を入力した状態で撮影：

| フィールド | 入力値 |
|----------|--------|
| 日付 | 今日の日付 |
| オンライン/オフライン | オフライン |
| 場所 | 渋谷カフェ |
| 内容 | 新規プロジェクトの件を相談。来月までに提案書を送る予定。次回は4月末に再度MTGを設定する。 |

## App Store Connect へのアップロード

1. [appstoreconnect.apple.com](https://appstoreconnect.apple.com) にログイン
2. アプリ選択 → 「App Store」タブ → 「6.9インチ」欄にドラッグ＆ドロップ
3. 順序を確認して保存
