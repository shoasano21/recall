# App Store 掲載文

## アプリ名
Recall

## サブタイトル（30文字以内）
大切な人のことを、もっと覚えていよう

## 説明文（400文字程度）

新しい職場、異業種交流会、旅先で出会った人——気づけば顔は浮かぶのに、名前も話した内容も忘れてしまっていませんか？

Recall は、出会った人のことを記録しておくための、シンプルな人脈メモアプリです。

名前・所属・趣味・出身地など基本情報を登録して、会うたびに会話の内容をログとして残せます。タグで整理すれば、「同期」「取引先」「勉強会」など自分だけのグループ管理も簡単です。

「次に会う日」を登録しておけば、前日夜に通知でリマインド。大切な場面で相手のことをスマートに思い出せます。

すべてのデータはあなたのデバイスにのみ保存されます。ログイン不要・広告なし・完全無料でお使いいただけます。
